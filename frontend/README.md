# frontend

Vite + React + Tailwind 4. Plain JavaScript, no TypeScript (see
[../docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md)).

## Running locally

```
npm install
npm run dev
```

Design tokens are defined in `src/index.css` (`@theme`) per
[../docs/UI-DESIGN.md](../docs/UI-DESIGN.md). Components use those tokens
only — no hardcoded colors.
